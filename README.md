## ✨ قابلیت‌ها

- 💬 **گفتگوی آزاد** - مکالمه طبیعی با هوش مصنوعی (GPT-4o, Claude, Llama)
- 🌐 **ترجمه متن** - ترجمه دقیق فارسی ↔ انگلیسی و سایر زبان‌ها
- 📝 **خلاصه‌سازی** - خلاصه کردن متون طولانی تا ۹۰٪
- 💻 **کمک کدنویسی** - نوشتن کد، دیباگ و توضیح مفاهیم برنامه‌نویسی
- 🎨 **ایده‌پردازی** - تولید ایده‌های خلاقانه برای کسب‌وکار و محتوا
- 🧠 **حافظه مکالمه** - به خاطر سپردن تاریخچه گفتگو تا ۱۰ پیام

## 🛠️ تکنولوژی‌ها

| تکنولوژی | کاربرد |
|----------|--------|
